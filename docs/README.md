# docs/

Additional design notes, decision records, and reference material for Free Humanoid Wheeled, beyond the canonical [README.md](../README.md), [ARCHITECTURE.md](../ARCHITECTURE.md), and [CONTRIBUTING.md](../CONTRIBUTING.md) at the root.

This directory will hold:

- **Architectural decision records (ADRs).** One file per resolved TBD. Each ADR cites the corpus entries that shielded the decision and links into [ARCHITECTURE.md §9](../ARCHITECTURE.md#9-architectural-commitments).
- **Mission-integration notes.** Detailed protocol-level documentation for the shoal cartridge-swap, waste-hopper retrieval, biology-reservoir handshake, and EAM REST API. Cross-linked to [clean_fish/ARCHITECTURE.md](../../clean_fish/ARCHITECTURE.md).
- **Operational concept.** Day-in-the-life of a deployed platform at a shoal dock-B site — work-order intake, transit, dock approach, cartridge-swap, biology-reservoir status, return-to-charge, fleet-coordination idle.
- **Indigenous and community consent protocol notes.** Same posture as `clean_fish` ARCHITECTURE §9 and the bipedal sibling.
- **Open questions.** Tracked TBDs from [ARCHITECTURE.md](../ARCHITECTURE.md) and any new questions raised by Phase 1 simulation.
